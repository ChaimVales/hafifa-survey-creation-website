# 📋 Survey Builder - מפרט פונקציונליות / Specification

## 🎯 תיאור הפרויקט

מערכת ליצירת סקרים בסגנון Google Forms. משתמשים יכולים להירשם, ליצור סקרים, לקבל URL ייחודי ולשתף, ולקבל תשובות גם ממשתמשים שאינם רשומים.

---

## 👥 משתמשים ותפקידים

| תפקיד | יכולות |
|-------|--------|
| **אורח (Guest)** | ✅ ענייה על סקר דרך לינק | ✅ דפדוף ברשימה ציבורית | ✅ הרשמה |
| | ❌ יצירת סקר | ❌ צפייה בתוצאות |
| **משתמש רשום (User)** | ✅ הכל של אורח | ✅ יצירת סקרים |
| | ✅ צפייה בתוצאות שלי | ✅ מחיקת סקרים שלי |
| | ❌ צפייה בסקרים של אחרים |

---

## 📝 User Stories

### אימות (Auth)

1. **כאורח, אני רוצה להירשם** עם אימייל וסיסמה, **כדי שאוכל ליצור סקרים משלי.**

2. **כמשתמש קיים, אני רוצה להתחבר** עם אימייל וסיסמה, **כדי לגשת לסקרים שלי.**

3. **כמשתמש מחובר, אני רוצה להתנתק**, **כדי להגן על החשבון שלי במחשב משותף.**

### יצירת סקרים

4. **כמשתמש רשום, אני רוצה ליצור סקר** עם כותרת, תיאור ושאלות, **כדי לאסוף תשובות מקהל.**

5. **כיוצר סקר, אני רוצה להוסיף שאלות מסוגים שונים** (טקסט, בחירה בודדת, בחירה מרובה), **כדי לקבל סוגי תשובות מגוונים.**

6. **כיוצר סקר, אני רוצה לסמן שאלות כחובה**, **כדי לוודא שאני מקבל תשובות מלאות.**

7. **כיוצר סקר, אני רוצה לקבל URL ייחודי**, **כדי לשלוח אותו בוואטסאפ או ברשתות חברתיות.**

### מענה על סקרים

8. **כאורח (לא מחובר), אני רוצה לענות על סקר דרך לינק**, **בלי להירשם.**

9. **כאורח, אני רוצה לדפדף ברשימה ציבורית של סקרים**, **כדי למצוא סקרים מעניינים לענות עליהם.**

10. **כעונה, אני רוצה לראות שגיאה אם פיספסתי שדה חובה**, **כדי לדעת מה להשלים.**

11. **כעונה, אני רוצה לקבל אישור אחרי השליחה**, **כדי לדעת שהתשובה נקלטה.**

### תוצאות

12. **כיוצר סקר, אני רוצה לראות תוצאות אגרגטיביות** (סיכום כולל אחוזים), **כדי להבין מה ענו רוב המשיבים.**

13. **כיוצר סקר, אני רוצה לראות גם תשובות פרטניות** (טקסט חופשי), **כדי לקרוא מה אנשים כתבו.**

### אבטחה

14. **כיוצר סקר, אני רוצה שרק אני אראה את התוצאות שלי** (לא יוצר סקר אחר), **כדי לשמור על פרטיות.**

15. **כיוצר סקר, אני רוצה למחוק סקר ישן**, **כדי לנקות את הדשבורד.**

## 📊 סוגי שאלות נתמכים

| סוג | שם בעברית | שם באנגלית | דוגמה |
|-----|-----------|------------|-------|
| `text` | טקסט קצר | Short Text | "שמך?" |
| `textarea` | טקסט ארוך | Long Text | "הערות נוספות?" |
| `single` | בחירה בודדת | Single Choice | "מין: זכר/נקבה/אחר" |
| `multi` | בחירה מרובה | Multi Choice | "מה אתה אוהב? פיצה/שניצל/חומוס" |

**מבנה שאלה:**
```typescript
{
    id: string          // nanoid אוטומטי (8 תווים)
    type: 'text' | 'textarea' | 'single' | 'multi'
    title: string       // הטקסט המוצג
    required: boolean   // ברירת מחדל: false
    options?: string[]  // רק ל-single/multi
}
```

## 🌐 דפי המערכת

| # | URL | דף | מוגן? |
|---|-----|-----|------|
| 1 | `/` | דף הבית | ❌ |
| 2 | `/login` | התחברות | ❌ |
| 3 | `/register` | הרשמה | ❌ |
| 4 | `/browse` | דפדוף בסקרים (חיפוש/מיון) | ❌ |
| 5 | `/s/:slug` | מענה על סקר | ❌ (ציבורי!) |
| 6 | `/dashboard` | הסקרים שלי | ✅ |
| 7 | `/surveys/new` | יצירת סקר | ✅ |
| 8 | `/surveys/:id/results` | תוצאות (רק הבעלים) | ✅ + Owner |

## ⚠️ מקרי קצה (Edge Cases) שטופלו

| תרחיש | פתרון |
|-------|-------|
| משתמש לוחץ "פרסם" עם 0 שאלות | ולידציה: 400 "missing questions" |
| שאלת choice ללא אופציות | ולידציה: 400 "need at least 2 options" |
| משתמש עונה על סקר שנמחק | 404 "Survey not found" |
| משתמש זר מנסה לראות תוצאות | 403 "Not your survey" |
| Token פג תוקף | Logout אוטומטי + redirect ל-login |
| שדה חובה ריק בתשובה | ולידציה client + server |
| URL של סקר עם slug לא חוקי | 404 בדף ידידותי |
| XSS בכותרת סקר | React escape אוטומטית ב-JSX |
| Duplicate email בהרשמה | 409 Conflict |
| סיסמה קצרה מ-6 תווים | ולידציה: 400 |
| Email לא תקין | ולידציה: 400 |

## 🎁 פיצ'רים נוספים שמומשו

✨ **דפדוף ציבורי** עם חיפוש (debounced) ומיון<br>
✨ **גרפי תוצאות** ויזואליים עם אחוזים<br>
✨ **העתקה ללוח** של URL בקליק (clipboard API)<br>
✨ **Loading states** בכל מקום<br>
✨ **טיפול שגיאות** עם הודעות ברורות<br>
✨ **RTL מלא** לעברית<br>
✨ **Cascade delete** - מחיקת סקר מוחקת גם את התשובות<br>
✨ **Dual environment** - Docker mongo או Atlas
